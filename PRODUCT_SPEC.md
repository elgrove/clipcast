# Clipcast Features

Clipcast creates ad-free podcast feeds from normal podcast feeds.

## Features

- Add podcasts to a personal library.
- Import podcast details, artwork, and episodes from source feeds.
- Keep podcast episode lists up to date.
- Show all saved podcasts in a library view.
- Show each podcast's episodes, dates, durations, and processing status.
- Choose a clipping mode per podcast: AI detection, Acast ident detection, or off.
- Use AI to analyse podcast episodes and identify advert sections.
- Use AI-generated timing information to decide which parts of the audio should be removed.
- For Acast-distributed podcasts, detect ad breaks via audio fingerprinting of the Acast sonic ident, without using AI transcription or token spend.
- Automatically suggest Acast clipping mode when adding a podcast whose feed is hosted on Acast.
- Remove detected advert sections from episode audio.
- Create edited audio files for processed episodes.
- Publish a separate ad-free RSS feed for each podcast.
- Let users subscribe to the ad-free feeds in any podcast app.
- Process individual episodes on demand.
- Process multiple episodes in a batch.
- Automatically process new episodes for selected podcasts.
- Show whether an episode is waiting, processing, complete, failed, or cleaned up.
- Show reports for recent processing runs.
- Allow podcast-specific AI instructions for advert detection.
- Allow old local episode files to be cleaned up.
- Export podcast subscriptions as OPML.
- Provide basic app settings for AI processing, appearance, and feed management.
